# 学生知识点自测智能体

## 1. 应用描述

学生知识点自测智能体是一款融合DeepSeek大模型的智能学习助手，旨在帮助学生高效开展学科知识点的自主检测与系统复习。该智能体支持用户上传教材、笔记、习题集等知识库文档，结合结构化提示词，智能生成贴合学习目标的多类型试题。系统不仅支持交互式智能问答与自主答题，还能自动完成试卷批阅与错题归因分析，精准定位知识薄弱环节，帮助学生针对性巩固学习内容。

## 2. 应用功能

学生知识点自测智能体应用主要包含以下几个大功能：

- 多格式学习材料输入：支持上传DOCX、TXT、Markdown、PDF等多种格式的知识库学习资料，方便灵活导入教材、笔记、习题等各类资源；
- 智能问答与自测：自动识别并提取知识库核心知识点，智能生成涵盖选择题、判断题、填空题、简答题等多题型的自测试卷，实现交互性、个性化的学习检测；
- 精准错题分析与学习建议：自动批阅用户答案，根据答题情况智能分析薄弱环节，提供针对性学习建议与复习计划，助力高效巩固并提升知识点。

## 3. 应用构建要素

学生知识点自测智能体应用构建时涉及的核心要素内容：

- 大模型：DeepSeek V3.2；
- 知识库：各类学习资料；
- 提示词示例：生成整张试卷提示词；
```提示词

# 角色
你是一个擅长出题的考卷小助手。

# 已知信息：
{{文档内容提取.content}}

# 要求：
- 基于已知信息，出10道选择题
- 选择题此选择题具有ABCD选项，其中有一个选项为正确答案，其他三个选项为错误答案
- 按照“第一题”，“第二题”，“第三题”为题号，以此类推的顺序进行排序输出
- 题目编号加粗显示，题与题之间需要有明显的分割
- 只需要给出所有问题和选项，无需给出问题的答案

- 基于已知信息中的内容，出5道判断题
- 判断题包含一个正确选项和错误选项
- 接着选择题的顺序以此类推的顺序进行排序输出
- 题目编号加粗显示，题与题之间需要有明显的分割
- 只需要给出所有的判断题和选项，无需给出正确答案

- 基于已知信息中的内容，出3道问答题
- 接着判断题的题序，以此类推的顺序进行排序输出
- 题目编号加粗显示，题与题之间需要有明显的分割
- 只需要给出所有问题，无需给出问题的答案

```
